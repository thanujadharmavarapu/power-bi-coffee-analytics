# Coffee Sales Insights Dashboard

**Project:** Microsoft Elevate Internship – Power BI  
**Author:** Lakshmi Thanuja Dharmavarapu  

---

## Project Overview
This project is an **interactive Power BI dashboard** built during the Microsoft Elevate Internship. It analyzes a coffee shop’s sales data, providing insights into customer behavior, spending patterns, and peak hours. The dashboard uses **DAX measures, interactive visualizations, and slicers** to explore coffee types, weekday vs. weekend trends, and daily sales performance.  

---

## Key Features
- **Total Spend & Total Cups Sold** – Track overall revenue and units sold.  
- **Peak Hour Analysis** – Identify hours with the highest sales volume.  
- **Weekday vs. Weekend Spend** – Compare customer spending patterns across the week.  
- **Average Spend Metrics** – Understand average spend per day and per weekend.  
- **Interactive Visualizations** – Line charts, clustered columns, donut charts, and KPI cards.  
- **Coffee Type Slicer** – Filter insights based on coffee products.  
- **Professional Formatting** – Consistent fonts, colors, and data labels for readability.  

---

## Technologies Used
- **Power BI Desktop** – Dashboard creation and interactive reporting.  
- **DAX (Data Analysis Expressions)** – Calculating measures for spend, cups sold, percentages, and peak hours.  
- **CSV Data Source** – Coffee shop sales data.  
- **Visualization Features** – KPI cards, line charts, column charts, donut charts, slicers, and buttons.  
---
## How to Use
1. Download the `CoffeeDashboard.pbix` file.  
2. Open it with **Microsoft Power BI Desktop**.  
3. Explore interactive visualizations and filters:  
   - Use the coffee type slicer to filter KPIs.  
   - View sales trends over time using line charts.  
   - Analyze weekday vs. weekend spending with column charts.  
4. Hover over charts to view detailed insights.  

---

## Insights Delivered
- Customers spend the most during **peak hours**, guiding staffing and promotions.  
- Spending patterns differ between **weekdays and weekends**, enabling targeted marketing.  
- Identifying **best-selling coffee types** helps optimize inventory.  
- KPI cards provide **quick metrics for management reporting**.
